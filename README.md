# FFmpeg Easy Converter

FFmpeg を使って、動画・音声・画像ファイルの変換を簡単に行うための  
Windows 向け GUI ツールです。

コマンドライン操作が必要な FFmpeg を、  
できるだけ手軽に扱えるようにすることを目的としています。

## 概要

- FFmpeg を使用したファイル変換 GUI
- Windows 専用

変換元ファイルと変換後ファイルを指定し、  
「変換」ボタンを押すだけで処理が実行されます。

## 使い方

1. 「変換元ファイル」ボタンから変換元ファイルを選択  
2. 「変換後ファイル」ボタンから変換後ファイルの保存先とファイル名、コンテナ形式を指定  
3. 「変換」ボタンを押す  

## FFmpeg のインストール方法（Windows）

本ソフトを使用するには、FFmpeg がインストールされており、  
`ffmpeg` コマンドに PATH が通っている必要があります。  

PowerShell を開き、以下を実行してください。

> winget install ffmpeg

インストール後、次のコマンドで動作確認ができます。

> ffmpeg -version

バージョン情報が表示されれば、正しくインストールされています。
